# 슬라이스
* go에서는 일반 배열보다 슬라이스를 더 많이 씀
* 슬라이스를 전달시, '슬라이스 헤더(포인터, 길이, 용량)'가 복사되어 전달됨.
    * 배열은 포인터'만' 전달됨
* 대체로 `배열선언` > `make([]int, 0, length)` >> `make([]int, 0, length)` 순으로 빠르다. [링크](discussion2.md)

```go
s := make([]int, length)
s := make([]int, 0, length) // 0 초기화를 안하는 배열 만들기
```
